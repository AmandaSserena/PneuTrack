
# Executar em 1 passo

## Windows
1) Dê 2 cliques em **run_windows.bat**  
   - Cria venv, instala dependências, inicializa o banco e **sobe o servidor**.

## macOS/Linux
```bash
chmod +x run_unix.sh
./run_unix.sh
```
